Instrução de Usabilidade

- Defina o padrão das colunas de metadados:

Ex.: Se for definido que o dado deve ser requerido, único, numérico.
O programa irá analisar e voltar para o usuário dados que cumprem esses requisitos.


- Se preciso, para uma otimização de busca, utilize o campo METADADOS para que o programa realize a busca no metadado selecionado com base nos requisitos definidos anteriormente.

Ex.: COD_MDL, CLI_CAD_POS, ID_STG_MDL


- Em descrição digite exatamente o dado deseja buscar para otimizar ainda mais sua análise.

Ex.: C01


Após todas as informações serem definidas. Clique em Confirmar ou cancelar para salvar ou cancelar as definições respectivamente.
